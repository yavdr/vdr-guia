# GUIA

## Requirements

- nodejs
- mongodb
- Active internet connection

## Installation
    git clone git://github.com/kersten/vdr-guia.git ./GUIA
    cd GUIA
    npm update

## Configuration
To configure GUIA just start it and point your browser to https://your.ip:8007.
